## 5. 查询分解流程详细图

```mermaid
sequenceDiagram
    participant User
    participant Graph
    participant MasterAgent
    participant LLM
    participant SearchAgent
    
    User->>Graph: 输入研究主题
    Graph->>MasterAgent: decompose_topic()
    
    MasterAgent->>MasterAgent: 准备 query_writer_instructions
    Note over MasterAgent: 包含:<br/>- TIME_CONTEXT<br/>- DATABASE_CONTEXT<br/>- AUGMENT_KNOWLEDGE_CONTEXT<br/>- STEERING_INSTRUCTIONS
    
    MasterAgent->>LLM: 调用 LLM 进行查询分解
    LLM-->>MasterAgent: 返回 JSON 分解结果
    
    MasterAgent->>MasterAgent: 解析 JSON<br/>{topic_complexity, subtopics}
    
    MasterAgent->>Graph: 返回分解结果
    Graph->>SearchAgent: 执行子任务
    
    loop 每个子任务
        SearchAgent->>SearchAgent: 选择合适的工具
        SearchAgent->>SearchAgent: 执行搜索/查询
        SearchAgent-->>Graph: 返回结果
    end
```

## 6. 报告生成和反思流程

```mermaid
sequenceDiagram
    participant Graph
    participant GenerateReport
    participant LLM1
    participant ReflectReport
    participant LLM2
    participant State
    
    Graph->>GenerateReport: 调用 generate_report 节点
    
    GenerateReport->>GenerateReport: 准备 summarizer_instructions
    Note over GenerateReport: 包含:<br/>- 搜索结果<br/>- 当前摘要<br/>- 研究主题<br/>- 引用要求
    
    GenerateReport->>LLM1: 调用 LLM 生成报告
    LLM1-->>GenerateReport: 返回综合报告
    
    GenerateReport->>GenerateReport: 后处理（添加引用）
    GenerateReport->>State: 更新 running_summary
    
    State->>ReflectReport: 调用 reflect_on_report 节点
    
    ReflectReport->>ReflectReport: 准备 reflection_instructions
    Note over ReflectReport: 包含:<br/>- 当前报告<br/>- 任务队列<br/>- 用户引导消息
    
    ReflectReport->>LLM2: 调用 LLM 评估报告
    LLM2-->>ReflectReport: 返回评估结果<br/>{knowledge_gap, next_query, research_complete}
    
    ReflectReport->>State: 更新任务队列和状态
    
    alt 研究未完成
        State->>Graph: 继续研究循环
    else 研究完成
        State->>Graph: 进入 finalize_report
    end
```

## 8. Steering 系统工作流

```mermaid
sequenceDiagram
    participant User
    participant API
    participant State
    participant TodoManager
    participant MasterAgent
    participant ReflectNode
    
    User->>API: 发送引导消息
    API->>State: add_steering_message()
    State->>TodoManager: 添加用户消息到队列
    
    Note over State,TodoManager: 消息存储在 pending_messages
    
    State->>MasterAgent: 执行研究
    MasterAgent->>MasterAgent: 读取 steering_todo.get_todo_md()
    MasterAgent->>MasterAgent: 根据 todo 调整查询生成
    
    MasterAgent-->>State: 完成研究循环
    
    State->>ReflectNode: reflect_on_report
    ReflectNode->>TodoManager: 获取待处理消息
    TodoManager-->>ReflectNode: 返回 pending_messages
    
    ReflectNode->>ReflectNode: LLM 处理消息<br/>创建/更新任务
    ReflectNode->>TodoManager: 更新任务状态
    Note over ReflectNode,TodoManager: - 标记完成任务<br/>- 添加新任务<br/>- 取消无关任务
    
    TodoManager->>State: 更新 todo 状态
    State-->>User: 返回更新后的计划
```

## 9. 完整调用时序图

```mermaid
sequenceDiagram
    participant Client
    participant API
    participant Graph
    participant State
    participant MasterAgent
    participant SearchAgent
    participant Tools
    participant LLM
    
    Client->>API: POST /research {topic, config}
    API->>State: 创建 SummaryState
    API->>Graph: 创建图实例
    API->>Graph: ainvoke(state, config)
    
    Graph->>MasterAgent: multi_agents_network()
    MasterAgent->>LLM: decompose_topic()<br/>使用 query_writer_instructions
    LLM-->>MasterAgent: 返回查询分解结果
    
    MasterAgent->>MasterAgent: 创建研究计划
    
    par 并行执行子任务
        MasterAgent->>SearchAgent: execute(subtask1)
        SearchAgent->>Tools: general_search(query1)
        Tools-->>SearchAgent: 搜索结果1
    and
        MasterAgent->>SearchAgent: execute(subtask2)
        SearchAgent->>Tools: academic_search(query2)
        Tools-->>SearchAgent: 搜索结果2
    and
        MasterAgent->>SearchAgent: execute(subtask3)
        SearchAgent->>Tools: text2sql(query3, db_id)
        Tools-->>SearchAgent: 数据库结果3
    end
    
    SearchAgent-->>MasterAgent: 所有结果
    MasterAgent->>State: 更新 web_research_results
    MasterAgent-->>Graph: 返回更新后的 state
    
    Graph->>Graph: generate_report()
    Graph->>LLM: 使用 summarizer_instructions
    LLM-->>Graph: 返回综合报告
    Graph->>State: 更新 running_summary
    
    Graph->>Graph: reflect_on_report()
    Graph->>LLM: 使用 reflection_instructions
    LLM-->>Graph: 返回评估结果
    
    alt 需要继续研究
        Graph->>Graph: route_research() → multi_agents_network
        Note over Graph: 循环继续
    else 研究完成
        Graph->>Graph: finalize_report()
        Graph->>LLM: 使用 finalize_report_instructions
        LLM-->>Graph: 返回最终报告
        Graph-->>API: 返回最终结果
        API-->>Client: {markdown_report, sources, ...}
    end
```
